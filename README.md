# AI-APPS

AI-powered Next.js applications with completion and streaming chat interfaces.

## Features

- **Completion** (`/ui/completion`) - One-shot AI text generation
- **Streaming** (`/ui/stream`) - Real-time streaming AI responses

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

- **Completion**: [http://localhost:3000/ui/completion](http://localhost:3000/ui/completion)
- **Streaming**: [http://localhost:3000/ui/stream](http://localhost:3000/ui/stream)

## Tech Stack

- [Next.js](https://nextjs.org)
- [Vercel AI SDK](https://sdk.vercel.ai)
- [OpenAI](https://openai.com)
